
	É utilizado o software chamado "Tiled", um editor de mapa para jogos tiled-based. Foi usado a biblioteca pytmx que instala-se brevemente na consola(CMD).

	Caso esteja a faltar essa biblioteca, para instalar na consola, basta digitar: pip install pytmx


	Todas as imagens e musicas foram retiradas do site opengameart.org, um site onde artistas oferecem artes de jogos, sem perigo de copyright.
